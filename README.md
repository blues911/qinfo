# qinfo

Displays simple system information (in real time) based on general Linux commands.

Available data:
- Distribution release
- Kernel version
- Uptime
- Load average
- CPU
- RAM
- HDD

## Install

<pre>
git clone https://github.com/smokehill/qinfo.git
cd qinfo/
make install
qinfo
</pre>
